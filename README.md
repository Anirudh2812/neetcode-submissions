# NeetCode Solutions

A collection of Data Structures and Algorithms solutions solved through NeetCode and synchronized automatically to GitHub.

---

## About

This repository contains my solutions to problems from NeetCode.io. The purpose of this repository is to:

* Improve problem-solving skills
* Practice common coding interview patterns
* Track learning progress over time
* Build consistency in Data Structures and Algorithms

Problems cover a wide range of topics, including:

* Arrays & Hashing
* Two Pointers
* Sliding Window
* Stack
* Binary Search
* Linked Lists
* Trees
* Heap / Priority Queue
* Backtracking
* Graphs
* Dynamic Programming
* Greedy Algorithms

## Repository Structure

Solutions are organized by topic and problem identifier.

```text
<topic-folder>/
 └── <problem-id>/
      ├── submission-0.<ext>
      ├── submission-1.<ext>
      └── ...
```

Example:

```text
Data Structures & Algorithms/
 ├── two-integer-sum/
 │    └── submission-0.py
 ├── binary-search/
 │    └── submission-0.py
 └── valid-anagram/
      └── submission-0.py
```
